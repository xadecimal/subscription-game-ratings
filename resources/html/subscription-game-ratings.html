Subscription Game Ratings

This page lists all current PS+ games that are part of the game catalog (extra and premium) and classics catalog (premium only) sorted by IGDB rating.

It gets the current list of PS+ games from Push Square and for each one retrieves the IGDB total rating (critics + users) and user rating.

The list is ranked by IGDB total rating, but also shows the Push Square score and user score along with the IGDB user rating.

Game Pass console and PC games will also be added eventually.
